それでは最初に、Unity 公式の UI Toolkit サンプルプロジェクトを幾つか紹介しましょう。
まず最初に紹介するのは Dragon Crashers UI Toolkit Sample というプロジェクトです。
これは Unity に以前から存在していた Dragon Crashers というサンプルプロジェクトを UI Toolkit 用に拡張したものです。
本格的な 2D ゲームの UI を実装したものになっています。
キャラクターのインベントリ画面や、ショップ画面、お知らせ画面、コンフィグ画面など、最近のモバイルゲームに定番の UI を一通り実装しています。
また UI のレイアウトはレスポンシブな設計になっていて、解像度やアスペクト比を動的に変更することができます。
UI のテーマも複数用意されていて、スイッチで切り替えられるようになっています。
季節のイベントに合わせて UI の見た目を変える、ということが、これでできるわけですね。
このように、この Dragon Crashers は、UI Toolkit のサンプルであると同時に、ゲーム UI として必要とされる部品を一通り揃えたデモンストレーションとなっています。
ただ、Dragon Crashers は、サンプルとしてはちょっと複雑過ぎるところがあるので、入門用の教材としては、もっとシンプルなプロジェクトを使った方がいいかもしれません。
そこで次に紹介するのが QuizU というプロジェクトです。
これは Unity を題材にした簡単なクイズゲームを実装したもので、画面上の表示はすべて UI Toolkit で作られています。
Dragon Crashers と比較すると構造がシンプルなので、UI Toolkit の基礎を学習するのに適しています。
また、本編のゲーム画面だけでなく、UI Toolkit の各種の機能を紹介する画面も用意されており、教材として役立つ内容になっています。
UI Tookit を実際に使い始めてから、各機能の使い方に迷った場合、この QuizU を参考にすることが多いですね。
UI Toolkit 学習の際には、常に手元に置いておきたいプロジェクトです。
ただ、このプロジェクトでも、最初の一歩目としては、まだ複雑過ぎるかなと思っています。
今日はこれよりも更に簡単な、超シンプルなサンプルから始めたいと思います。
